====================================================
 WP ThemingStrap - Bootstrap 3 Responsive WordPress Theme
====================================================

Theming Strap is a free version of ThemingStrap Framework made for premium quality theme development. This theme is having a lot of theme functionality like manage custom logo, Fav Icon, Footer script, Social media profile etc. It is fully Powered by Bootstrap 3.0 and FontAwesom icon font. Visit http://themingstrap.com/ to get details about this theme and framework. This theme is built using code standards of twentytwelve theme & like WordPress it is free to use for personal & commercial purpose.
